Typography Design System

A comprehensive guide to our modular type scale, designed for optimal readability on standard displays (1024x768 and up). All measurements use REMs for better accessibility and scaling.

Type Scale Overview

Our typography system consists of seven distinct styles, each serving specific purposes in the interface hierarchy. Base font size is 16px (1rem).

Primary Heading (Heading 1)

40px/48px • Bold • Use for main page titles

Section Heading (Heading 2)

32px/40px • Bold • Use for major section headers

Subsection Heading (Heading 3)

24px/32px • Bold • Use for subsections and feature headings

Body 1 - Large body text for important paragraphs and introductory content.

21px/32px • Regular • Use for hero text and key messages

Body 2 - Medium body text that works well for main content areas and longer form reading.

18px/28px • Regular • Use for main content

Body 3 - Standard body text suitable for general content and longer paragraphs where space efficiency is important.

16px/24px • Regular • Use for general content

Note - Small text for auxiliary information, captions, and metadata.

14px/21px • Regular • Use for supporting text

Usage Guidelines

Hierarchy

Maintain proper hierarchy by using styles in sequence. Don't skip levels (e.g., from H1 to H3) unless there's a specific design reason.

Spacing

Headers include bottom margins (0.6em-1em) to maintain consistent vertical rhythm. Adjust only when necessary for specific layouts.

Font Stack

Avenir Condensed is our primary typeface, with system fonts as fallbacks. This ensures consistent rendering across all platforms.

Implementation

Apply these styles using the provided utility classes:

        
  .heading-1, .heading-2, .heading-3 - Header styles
  .body-1, .body-2, .body-3 - Body text styles
  .note - Small text style

All measurements use REMs for better accessibility. Base font size is 16px (1rem).

Designing with Purpose

A comprehensive guide to our typography system, showcasing how different text styles work together to create clear visual hierarchy and improved readability.

Key Principles

Our typography system is built on three fundamental principles: hierarchy, readability, and consistency. Each text style serves a specific purpose in our design language.

Visual Hierarchy

Strong typographic hierarchy guides users through content, making information easy to scan and understand at a glance.

Readability

Carefully chosen font sizes and line heights ensure comfortable reading experiences across all device sizes.

Consistency

A systematic approach to typography helps maintain visual consistency throughout the user interface.

Implementation

Our typography system uses the Avenir Condensed font family, chosen for its excellent readability and modern character.

The system includes three header sizes and four text styles, providing flexibility for various content needs while maintaining consistency.

Note: All text styles are available as utility classes and can be easily applied to any element in your HTML.
